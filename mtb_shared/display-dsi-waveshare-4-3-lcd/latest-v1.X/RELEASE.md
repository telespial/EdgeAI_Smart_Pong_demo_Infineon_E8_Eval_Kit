# Waveshare 4.3-inch DSI display driver library release notes

This driver library provides functions for supporting a [Waveshare 4.3 inch DSI Capacitive Touch Display](https://www.waveshare.com/4.3inch-dsi-lcd.htm) on PSOC&trade; Edge E8x Evaluation Kit.

## What's included?

- APIs for initializing and de-initializing the driver
- APIs for backlight brightness control


See the [README.md](./README.md) and the [API reference guide](./API_reference.md) for a complete description of the display-dsi-waveshare-4-3-lcd.
