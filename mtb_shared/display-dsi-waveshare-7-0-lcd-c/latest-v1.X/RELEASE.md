# Waveshare 7-inch Raspberry Pi DSI C display driver library release notes

This is the driver library for interfacing the Waveshare 7-inch Raspberry Pi DSI C display with the PSOC&trade; Edge E84 Evaluation Kit. 

## What's included?

- APIs for initializing and de-initializing the driver
- APIs for backlight brightness control


See the [README](./README.md) and the [API reference guide](./API_reference.md) for a complete description of the display-dsi-waveshare-7-0-lcd-c driver library.
