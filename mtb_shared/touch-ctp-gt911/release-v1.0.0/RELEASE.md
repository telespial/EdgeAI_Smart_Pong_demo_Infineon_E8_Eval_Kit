# GT911 touch panel driver library release notes

This library provides a basic set of functions to support Capacitive Touch Panel (CTP) of Waveshare 7 inch DSI C display driven by [GT911](https://www.crystalfontz.com/controllers/datasheet-viewer.php?id=464) CTP controller. This Waveshare 1024 x 600 pixels 7 inch DSI C video mode display panel is used with PSOC&trade; Edge E84 Evaluation Kit.

## What's included?

 - APIs for initializing and de-initializing the driver
 - API to read single touch point coordinates 

See the [README](./README.md) and the [API reference guide](./API_reference.md) for a complete description of the GT911 touch panel Driver Library.
