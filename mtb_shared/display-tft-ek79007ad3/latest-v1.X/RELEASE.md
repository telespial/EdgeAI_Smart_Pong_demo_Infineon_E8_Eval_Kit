# EK79007AD3 TFT DSI display driver library release notes

This library provides functions for supporting a 10.1 inch TFT LCD driven by an EK79007AD3 controller. PSOC&trade; EDGE E84 Evaluation Kit supports this 10.1 inch 1024*600 pixel MIPI DSI TFT LCD [WF101JTYAHMNB0](https://www.winstar.com.tw/products/tft-lcd/ips-tft/ips-touch.html).

## What's included?

- APIs for initializing and de-initializing the driver
- APIs for backlight configuration and brightness control


See the [README.md](./README.md) and the [API reference guide](./API_reference.md) for a complete description of the display-tft-ek79007ad3.
